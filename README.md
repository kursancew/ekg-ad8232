# ekg-ad8232
AD8232 based ECG

AD8232 is a simple way to measure the electric current as it's transmitted through your heart.

Hookup guide: https://learn.sparkfun.com/tutorials/ad8232-heart-rate-monitor-hookup-guide

Datasheet: http://www.analog.com/media/en/technical-documentation/data-sheets/AD8232.pdf
